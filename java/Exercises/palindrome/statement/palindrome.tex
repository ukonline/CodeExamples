\section*{Contexte}

Un palindrome est une phrase qui se lit de la même manière dans les deux sens, sans prendre en compte les espaces, ni la ponctuation. On a par exemple les palindromes suivants :

\begin{quote}\it
ressasser, radar, engage le jeu que je le gagne
\end{quote}

%% - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

\section*{Implémentation}

Définir une méthode \code{isPalindrome} qui prend en paramètre une chaine de caractères et qui renvoie un booléen qui indique si la phrase est un palindrome (\code{true}) ou non (\code{false}), en n'utilisant que les méthodes \code{charAt} et \code{length} de la classe \code{String}.

Commencez par définir une méthode auxiliaire \code{clean} qui prend en paramètre une chaine de caractères, composée uniquement de lettres minuscules et d'espaces, et qui renvoie la même chaine sans tous les espaces qu'elle contient. Cette méthode vous sera ensuite utile pour définir la méthode \code{isPalindrome}.

\begin{lstlisting}[language=java]
public class Palindrome
{
	private static String clean (String s)
	{
		// À compléter
	}
	
	public static boolean isPalindrome (String s)
	{
		// À compléter
	}
	
	public static void main (String[] args)
	{
		String str = "engage le jeu que je le gagne";
		if (isPalindrome (str))
		{
			System.out.println ("'" + str + "' est un palindrome.");
		}
	}
}
\end{lstlisting}

\medskip
L'exécution de l'exemple ci-dessus doit afficher \code{'engage le jeu que je le gagne' est un palindrome.}

\section*{Astuces}

La méthode \code{length()} renvoie le nombre de caractères d'une chaine de caractères et la méthode \code{charAt~(i)} renvoie le caractère à l'indice \code{i} dans la chaine de caractères (sachant que le premier caractère est celui d'indice $0$).
