\section*{Contexte}

Cet exercice concerne la manipulation de fichiers textes. Soit un fichier texte qui contient, sur chacune de ses lignes, une séquences de nombres entiers, séparés par exactement un espace. Le but de l'exercice consiste à écrire un programme qui va lire ce fichier et calculer, pour chaque ligne, la somme des nombres de la ligne. Toutes ces sommes doivent être écrites dans un autre fichier texte. Si une ligne du fichier d'origine n'est pas valide, le programme devra écrire \code{ERREUR} comme résultat.

Voici un exemple concret d'un fichier \code{input.txt} et du résultat \code{output.txt} que doit créer votre programme :

\begin{center}
	\scalebox{0.8}{\begin{tabular}{|>{\tt}p{3cm}|}
		\hline
		12 7 9 -8 \\
		6 0 0 2 \\
		-8 -2 12 SEB \\
		\\
		1 \\
		1 2 3 quatre \\
		89 98 \\
		\hline
		input.txt \\
		\hline
	\end{tabular}}
	\hspace{2cm}
	\scalebox{0.8}{\begin{tabular}{|>{\tt}p{3cm}|}
		\hline
		20 \\
		8 \\
		ERREUR \\
		0 \\
		1 \\
		ERREUR \\
		187 \\
		\hline
		output.txt \\
		\hline
	\end{tabular}}
\end{center}

\section*{Implémentation}

Définir une méthode \code{computeSum} qui prend en paramètres deux chaines de caractères représentant des chemins vers des fichiers texte. Le première paramètre est le nom du fichier source à lire et le second paramètre est le fichier destination à créer et qui va contenir les sommes calculées par le programme.

En cas d'erreurs de lecture ou d'écriture de fichier (\code{IOException}), la méthode \code{computeSum} quitte immédiatement le programme en affichant un message d'erreur.

\begin{lstlisting}[language=java]
public class FileSum
{
	private static void computeSum (String inFile, String outFile)
	{
		// À compléter
	}
	
	public static void main (String[] args)
	{
		computeSum ("input.txt", "output.txt");
	}
}
\end{lstlisting}
